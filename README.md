# Coverletter

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

```bash
pnpm dev
pnpm build
pnpm start
pnpm lint
```

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- ESLint
